To make the android emulator work, make sure the following is done:

1. Open AVD Manager
2. Ensure, that a Virtual Device with the following settings exist:
	Hardware: Google Pixel 2
	System Image: Android 4.4 KitKat (API level 19)
3. In the AVD manager, press the drop-down menu.
4. COLD-BOOT the Virtual Device FROM THE AVD MANAGER.
5. In Android Studio, press run 'app'

Repeat step 3-5 each time.
